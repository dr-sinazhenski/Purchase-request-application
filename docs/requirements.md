# Purchase-request-application

## Description
A web application for managing internal purchase requests within a company.  
Users can create purchase requests, while approvers review and decide whether to approve or reject them.  
The system supports role-based access, request lifecycle management and working with multiple currencies.

## Target Audience
The application is intended for organizations of any size that implement internal procurement and approval processes.  
It is especially useful for companies that require managerial approval and budget control before executing purchases.
