## Roadmap

### MVP (к неделе 10)
- [ ] Авторизация пользователей
- [ ] Роли: Requester / Approver
- [ ] Разграничение прав доступа (RBAC)
- [ ] Создание запроса (Create)
- [ ] Просмотр списка запросов (Read)
- [ ] Редактирование запроса (Update)
- [ ] Удаление запроса (Delete)
- [ ] Базовый workflow (NEW -> APPROVED / REJECTED)
- [ ] Просмотр деталей запроса
- [ ] Подключение PostgreSQL
- [ ] Поиск и фильтрация запросов
- [ ] Сортировка (по цене, дате, пользователю)
- [ ] Причина отклонения запроса
- [ ] Повторная отправка (resubmit)
- [ ] Конвертация валют (multi-currency support)
- [ ] Мульти-товарные запросы (несколько позиций)
- [ ] Подсчёт total amount на основе позиций
- [ ] Профили согласования (approval profile)
- [ ] Аутентификация и авторизация (JWT)
- [ ] REST API документация (Swagger)
- [ ] Обработка ошибок и логирование
- [ ] Деплой (cloud / CI/CD)

### v1.0 (после курса)
- [ ] Добавление роли: Admin
- [ ] Несколько уровней согласования 
- [ ] Уведомления (email / in-app)
- [ ] История изменений (audit log)

### v2.0 (после курса)
- [ ] Автоматическое создание purchase order после одобрения
- [ ] Поддержка разных типов запросов (purchase, equipment, internal requests)
- [ ] Конфигурируемые типы запросов
- [ ] Унифицированная модель данных для всех типов заявок
- [ ] Гибкие workflow (разные сценарии согласования)
- [ ] Интеграция с системой учета/склада
